# Contributing

Thanks for your interest in this repository.

This is a **personal configuration project** used to set up my own development environments across macOS and Linux systems.
As such, I am **not accepting direct contributions or pull requests** that change behavior, tooling choices, or workflows.

## Forking

You are absolutely welcome to:
- Fork this repository
- Reuse scripts
- Adapt the configuration to your own needs

This repository is opinionated by design and reflects my personal preferences.

## Feedback & Suggestions

While direct contributions are not accepted, I **do welcome**:

- Suggestions for cleaner or more maintainable code
- Tips for improving shell scripts
- Best practices related to tooling or system setup
- Performance or reliability improvements

These can be shared via:
- GitHub Issues
- Discussions (if enabled)
- Comments on existing code
- Email

## No Guarantees

- Suggestions may or may not be adopted
- There is no roadmap or obligation to support external use cases

Thanks for understanding and for taking inspiration from the project.
